Selenium

Acceptance tests: These test-suites demonstrate/exercise the functionality of Selenium.

  • Main TestSuite - tests for all of the features that we consider core to Selenium
  • Internet Explorer TestSuite - tests for all core features, plus IE specific features
  • Konqueror TestSuite - tests that currently pass on Konqueror
  • Safari TestSuite - tests that currently pass on Safari
  • devtests - experimental features
  • Error Checking TestSuite - tests for the error verification commands
  • GoTo support - MetaCommunication's extension

Unit-tests: Use JsUnit to test Selenium internals.
