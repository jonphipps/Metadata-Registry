How to create a form that modifies an object
============================================

Overview
--------

Modifying an object from the data entered in a form is something that is very common in web programming, and a repetitive task. It is also a time consuming matter since the template design will have to take into account the hci and accessibility topics. Symfony helps to simplify this task with high level functions automating object modification from form data.
