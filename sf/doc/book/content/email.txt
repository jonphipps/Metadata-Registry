How to send an email
====================

Introduction
------------

Email sending is handled by the `sfMail` class. It is a proxy class that offers an interface with [PhpMailer](http://phpmailer.sourceforge.net/).

This proxy class allows you to define your own interface if required.

Example
-------

    [php]
    require_once 'sf/sfMail.class.php';
    
    // class initialization
    $mail = sfMail::newInstance('PhpMailer');
    $mail->initialize();
    $mail->setMailer('sendmail');
    
    // definition of the required parameters
    $mail->addAddress('client@client.com');
    $mail->setSender('me@symfony-project.com', 'Symfony');
    $mail->setFrom('me@symfony-project.com', 'Symfony');
    $mail->addReplyTo('you@symfony-project.com');
    $mail->setCharset('utf-8');
    $mail->setSubject('A very important matter');
    $mail->setBody('First of all, you should know that...');
    
    // send the email
    $mail->send();
